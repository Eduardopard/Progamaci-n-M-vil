## Cómo ejecutar el proyecto

1. Clonar el repositorio:

```bash
git clone https://github.com/TU_USUARIO/TU_REPOSITORIO.git
```

2. Entrar a la carpeta del proyecto:

```bash
cd eduardo_pardo_s1
```

3. Instalar dependencias:

```bash
flutter pub get
```

4. Ejecutar la aplicación:

```bash
flutter run
```
